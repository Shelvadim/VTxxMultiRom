# How to regenerate original_menu_patched.rom on your computer:

You need Python 3 (no third-party packages — pure stdlib). Then:

## 1. Open a terminal / command prompt in your project folder

## 2. Run the assembler:
python menu_assemble.py original_menu_patched.rom

## Or specify a different output path:
python menu_assemble.py "G:\Consoles\NES\Onebus Maker5\original_menu_patched.rom"
